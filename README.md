# 🛠️ Lua Patcher Website

A modern, high-performance web interface for the Lua Patcher tool. Built with React, Vite, and designed with a focus on speed, aesthetics, and ease of use.

![Lua Patcher Dashboard](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## ✨ Features

- **Dynamic Game Details**: Detailed information for patched games.
- **Steam Integration**: Ready for Steam API integration to fetch game data.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.
- **Blazing Fast**: Powered by Vite and React for near-instant load times.
- **Premium UI**: Sleek, modern interface using custom CSS and glassmorphism.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sayedalimollah2602-prog/lua-patcher-website.git
   cd lua-patcher-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Copy the example environment file and add your keys:
   ```bash
   cp .env.example .env.local
   ```
   Edit `.env.local` and add your `GEMINI_API_KEY` or other necessary keys.

4. **Launch development server:**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173`.
